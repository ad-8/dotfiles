* prompt
testing https://github.com/sindresorhus/pure

** 1. add to fpath
#+begin_src sh
fpath+=($HOME/.zsh/pure)
#+end_src

** 2. set opts
#+begin_src sh
export PURE_PROMPT_SYMBOL='λ'
export PURE_CMD_MAX_EXEC_TIME=2
#+end_src
   
** 3. init
Initialize the prompt system and choose pure:
#+begin_src sh
autoload -U promptinit; promptinit
prompt pure
#+end_src
